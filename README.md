# RecipeCloud

Aplicación web para guardar y consultar recetas de cocina.

## Objetivo

El objetivo del proyecto es crear una aplicación web sencilla donde los usuarios puedan almacenar, organizar y consultar recetas.

## Uso

El proyecto permite tener un listado de recetas con su nombre, autor, ingredientes y preparación.

## Tecnologías utilizadas

- Git
- GitHub
- GitHub Desktop
